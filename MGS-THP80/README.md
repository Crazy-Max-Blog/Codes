# Особенности:

| Параметр    | Описание |
| ----------- | -----------|
| Интерфейс   | I2C|
| Адрес       | 0x76 и 0x77 - в зависимости от перемычки JP0 |
| Питание     | 5В|
| Разъемы     | RJ-9, 2.54mm pin|
